Depedencies

Install Python > 3.5
pip install pycryptodome (to install python cryptography)

How to send command:-
1. To send a command. Send cmd (your command) ex:- cmd ls 
Server will respond ACK if task is complete. NOACK if task is incomplete. 

2. To send normal message. message ex:- Hiiiii.
Server will respond You send text message


Instruction:-
1. Open cmd in current directory.
2. Run Server type command python server.py
3. Run Single CLient type command python client.py
